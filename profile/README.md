# Zinc

A systems programming language that doesn't hate you.

Most systems languages give you control at the cost of spending half your
time thinking about memory instead of your actual problem. Zinc is an
experiment in whether that's actually a necessary tradeoff, or just an
assumption nobody questioned hard enough.

**Zinc is a systems language with:**
- Syntax that doesn't make you feel like you're filing paperwork
- A memory model that stays out of your way when it can
- The performance and control you actually need at the systems level

> ⚠️ Zinc is in early active development. Nothing here is stable.
> Watch this space.
